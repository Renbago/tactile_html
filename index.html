Tactile MPF — Interactive 3D Viewers

Per-probe inspection of the Shadow Hand + Manifold Particle Filter results on the external 768-grasp benchmark. Each viewer renders the predicted object pose, ground-truth pose, top-$K$ posterior modes, and active/inactive tactile sites in 3D.

Headline runs

All 768 probes are available, split into 4 parts of 192 probes each so that no single file exceeds the GitHub size limit.

How to use

Open any viewer in a modern browser (Chrome / Firefox / Edge / Safari). Use the probe dropdown to step through individual grasps; use the mouse to rotate the 3D scene. Argmax and posterior-mean parts share the same external 768-grasp test set in the same order, so the same graspqp_<N> probe can be compared side by side by opening the matching part on both sides.

Supplementary material for the thesis "Single-shot in-hand pose estimation of a known ellipsoid via manifold particle filter and contrastive learning" (SDU, 2026). The argmax vs posterior-mean comparison is the central ablation reported in Section: Results — Argmax vs posterior-mean cluster representative.
